# 使用方式

Maven

```xml
<dependency>
    <groupId>io.github.kongweiguang</groupId>
    <artifactId>kong-bus-springboot-starter</artifactId>
    <version>0.3</version>
</dependency>
```

Gradle

```
implementation 'io.github.kongweiguang:kong-bus-springboot-starter:0.1'
```

Gradle-Kotlin

```
implementation("io.github.kongweiguang:kong-bus-springboot-starter:0.1")
```
